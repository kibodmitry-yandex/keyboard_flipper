## Что это?

**Keyboard Flipper** - это автоматический конвертер раскладки клавиатуры. Он решает классическую проблему: вы писали текст, забыли переключить раскладку и получили вот это:

```
Вводили:  gthbvtnf pfhf
Хотели:   периметра зара
```

### Как работает?

1. **Выделите** неправильно набранный текст
2. **Нажмите** горячую клавишу (`Ctrl+Shift+F`)
3. **Готово!** Текст автоматически переведён в правильную раскладку

## Конфигурация

### 📄 Файл: `keyboard_flipper_config.json`

Находится рядом с `KeyboardFlipper.exe`

```json
{
  "hotkey": "ctrl+shift+f",
  "keyboard_layout": "ru_us"
}
```

### Как изменить горячую клавишу?

**Способ 1: Прямое редактирование**

1. Откройте `keyboard_flipper_config.json` в Notepad
2. Измените строку `"hotkey": "ctrl+shift+f"` на нужную
3. Сохраните файл
4. Перезагрузите приложение

**Пример - вместо `Ctrl+Shift+F` использовать `Alt+Z`:**

```json
{
  "hotkey": "alt+z",
  "keyboard_layout": "ru_us"
}
```

### Поддерживаемые горячие клавиши

| Сочетание | Как пишется |
|-----------|-----------|
| Ctrl+Shift+F | `ctrl+shift+f` |
| Alt+Z | `alt+z` |
| Ctrl+Alt+X | `ctrl+alt+x` |
| F12 | `f12` |
| Shift+F12 | `shift+f12` |
| Windows+K | `win+k` |
| Ctrl+Пробел | `ctrl+space` |

**Формат:**
- Модификаторы: `ctrl`, `alt`, `shift`, `win`
- Разделитель: `+`
- Строчные буквы

## FAQ

### ❓ Приложение требует права администратора?

**Ответ:** Да, для регистрации глобальных горячих клавиш нужны права администратора. Это нормально для любого такого приложения.

### ❓ Работает ли в любых программах?

**Ответ:** Да! Keyboard Flipper использует глобальные горячие клавиши, которые работают везде:
- ✅ Браузеры
- ✅ Текстовые редакторы
- ✅ Photoshop, Figma
- ✅ Messengers (WhatsApp Web, Telegram Desktop)
- ✅ Почтовые клиенты

**Исключение:** Приложения, работающие в режиме администратора, могут требовать дополнительных прав.

### ❓ Сколько памяти занимает?

**Ответ:** Около 8-10 MB во время работы. Очень легко.

### ❓ Что произойдёт с буфером обмена?

**Ответ:** Приложение сохраняет оригинальное содержимое буфера обмена и восстанавливает его после преобразования. Ваши данные не будут потеряны.

### ❓ Что если я сотру config.json?

**Ответ:** Приложение создаст новый с настройками по умолчанию при следующем запуске.

### ❓ Можно ли использовать несколько горячих клавиш?

**Ответ:** В текущей версии - только одна. Но вы можете:
1. Модифицировать `keyboard_flipper.py`
2. Добавить `keyboard.add_hotkey()` для других комбинаций
3. Пересобрать EXE

### ❓ Как добавить иконку в трей?

**Ответ:** В `keyboard_flipper.py` функция `create_icon()`:

```python
def create_icon(self) -> Image.Image:
    # Вместо текста "KF" можно загрузить картинку:
    img = Image.open("icon.png")
    return img
```

### ❓ Почему именно Ctrl+Shift+F?

**Ответ:** Это редко используемая комбинация, которая не конфликтует с большинством приложений. Но вы легко можете изменить её!

### ❓ Что если горячая клавиша не работает?

**Ответ:** Попробуйте:
1. Запустите приложение от администратора
2. Проверьте, что комбинация синтаксически правильная в config.json
3. Убедитесь, что эта клавиша не используется другим приложением

### ❓ На что потребляется 6.7 MB?

**Ответ:**
- Python runtime: ~5 MB
- Зависимости (pystray, pillow и т.д.): ~1.5 MB
- Сам код: ~50 KB
