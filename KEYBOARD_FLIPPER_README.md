# Keyboard Flipper

Портативное приложение для автоматического переключения раскладки клавиатуры. Раз и навсегда забудьте о проблеме "забыл переключить раскладку"!

## Возможности

✨ **Автоматическое определение раскладки**
- Анализирует выделенный текст и определяет основной язык
- Если большая часть текста на латинице → конвертирует в кириллицу
- Если большая часть текста на кириллице → конвертирует в латиницу

🎯 **Горячая клавиша**
- По умолчанию: `Ctrl+Shift+F`
- Работает в любых приложениях (браузер, Фотошоп, текстовые редакторы и т.д.)

🔧 **Персонализация**
- Настройки хранятся в `keyboard_flipper_config.json` рядом с приложением
- Легко менять горячую клавишу и тип раскладки

📦 **Портативность**
- Один файл `KeyboardFlipper.exe` - никакой установки
- Ноль зависимостей на систему
- Висит в системном трее, готово к использованию

## Использование

### Вариант 1: Запуск Python скрипта

```bash
# Установите зависимости
pip install keyboard pyperclip pystray pillow

# Запустите скрипт
python keyboard_flipper.py
```

### Вариант 2: Сборка в EXE

```bash
# Установите PyInstaller
pip install pyinstaller

# Соберите EXE
python build_exe.py
```

После сборки найдёте `KeyboardFlipper.exe` в папке `dist/`.

## Как это работает

1. **Скопируйте текст** или выделите его в любом приложении
2. **Нажмите** `Ctrl+Shift+F` (или настроенную горячую клавишу)
3. **Текст автоматически конвертируется** в нужную раскладку
4. **Результат вставляется** вместо оригинального текста

## Конфигурация

Отредактируйте `keyboard_flipper_config.json`:

```json
{
  "hotkey": "ctrl+shift+f",
  "keyboard_layout": "ru_us"
}
```

### Поддерживаемые горячие клавиши

- `ctrl+shift+f`
- `alt+shift+x`
- `ctrl+alt+f`
- `f12`
- И любые другие комбинации (см. документацию `keyboard` library)

## Архитектура

- `keyboard_flipper.py` - основное приложение с системным треем
- `build_exe.py` - скрипт для сборки в EXE
- `keyboard_flipper_config.json` - конфигурация (генерируется автоматически)

## Требования

**Для Python скрипта:**
- Python 3.7+
- keyboard
- pyperclip
- pystray
- pillow

**Для EXE:**
- Windows 7+
- Никаких внешних зависимостей

## Известные ограничения

- Требуются права администратора для регистрации глобальных горячих клавиш
- Некоторые приложения (например, приложения, работающие в режиме администратора) могут требовать дополнительных прав
