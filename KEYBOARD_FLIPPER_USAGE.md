# 🎹 Keyboard Flipper - Детальное руководство

## Быстрый старт

### Вариант 1️⃣: Запуск готового EXE (проще всего)

1. Откройте `run_keyboard_flipper.bat`
2. Приложение появится в системном трее (правый нижний угол)
3. **Готово!** Теперь можно использовать горячую клавишу `Ctrl+Shift+F`

### Вариант 2️⃣: Запуск Python скрипта (если хотите модифицировать)

```bash
python keyboard_flipper.py
```

## Как пользоваться

### Сценарий 1: Забыли переключить раскладку

Вы находитесь в браузере, писали сообщение на русском, но забыли переключить раскладку:

```
Вводили:     gthbvtnf
Хотите:      периметра
```

**Что делать:**
1. Выделите неправильно набранный текст `gthbvtnf` (кликните-перетащите или двойной клик)
2. Нажмите `Ctrl+Shift+F`
3. Текст автоматически заменится на `периметра` ✨

---

### Сценарий 2: Смешанная раскладка

Вы набирали текст, иногда переключая раскладку:

```
Вводили:     Привет! Hello мир
Выделяете:   Привет! Hello мир
```

**Что происходит:**
- Приложение анализирует текст: находит **больше кириллицы** (~60%)
- Конвертирует **весь текст** в кириллицу
- Результат: `Ghbdtn! Уnkk vbh`

---

### Сценарий 3: Профессиональное использование

Используете **Photoshop** или другой графический редактор:

1. Выделяете текст в инструменте текста
2. `Ctrl+Shift+F` → текст мгновенно переворачивается
3. Продолжаете работу

## Конфигурация

### Файл: `keyboard_flipper_config.json`

```json
{
  "hotkey": "ctrl+shift+f",
  "keyboard_layout": "ru_us"
}
```

### Доступные горячие клавиши

Вместо `ctrl+shift+f` можно установить:

| Сочетание | Описание |
|-----------|---------|
| `ctrl+shift+c` | Ctrl+Shift+C |
| `alt+shift+x` | Alt+Shift+X |
| `ctrl+alt+f` | Ctrl+Alt+F |
| `f12` | Только F12 |
| `shift+f12` | Shift+F12 |
| `ctrl+shift+space` | Ctrl+Shift+Пробел |

**Синтаксис горячих клавиш:**
- Модификаторы: `ctrl`, `alt`, `shift`, `win`
- Разделитель: `+`
- Пример: `ctrl+shift+alt+k`

### Типы клавиатуры

В `keyboard_layout` могут быть:
- `ru_us` - русский ↔ английский (по умолчанию, дёво-кв)
- Легко расширить для других раскладок

## Расширенное использование

### Автозагрузка при старте Windows

**Способ 1: Через батник (простой)**
```bash
setup_autostart.bat
```

**Способ 2: Ручной способ**
1. Скопируйте `dist/KeyboardFlipper.exe`
2. Откройте папку автозагрузки: `C:\Users\[ВАШ_ПОЛЬЗОВАТЕЛЬ]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup`
3. Вставьте там ярлык на приложение

### Изменение конфигурации

1. Откройте `keyboard_flipper_config.json` в текстовом редакторе
2. Отредактируйте горячую клавишу или раскладку
3. Сохраните файл
4. Перезапустите приложение

**Пример: изменить горячую клавишу на Alt+Z**
```json
{
  "hotkey": "alt+z",
  "keyboard_layout": "ru_us"
}
```

## Часто задаваемые вопросы

### ❓ Почему приложение требует права администратора?

Для регистрации глобальных горячих клавиш нужны повышенные права. Это нормально для любого такого приложения.

### ❓ Работает ли в Photoshop/Chrome/Word?

**Да!** Приложение использует глобальные горячие клавиши, работает везде. Но в некоторых приложениях (например если они работают в режиме администратора) может потребоваться запустить Keyboard Flipper также от администратора.

### ❓ Можно ли изменить горячую клавишу?

**Да!** Отредактируйте `keyboard_flipper_config.json`:
```json
{
  "hotkey": "f12",
  "keyboard_layout": "ru_us"
}
```

### ❓ Сколько памяти занимает?

Примерно **5-10 MB в памяти** во время работы (очень легко).

### ❓ Что если буфер обмена был заполнен?

Приложение сохраняет и восстанавливает оригинальное содержимое буфера обмена, поэтому ваши данные не потеряются.
